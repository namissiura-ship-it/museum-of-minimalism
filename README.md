# Museum of Minimalism — демо-сайт (Flask)

Сайт-музей по мотивам присланного макета (Figma): страница музея,
карточка выставки, планирование визита, бронирование билетов,
оформление заказа с оплатой (демо) и страница подтверждения.

## Стек
- **Python 3 + Flask** — маршруты, сессии, серверная логика (расчёт цены,
  промокоды, таймер резерва 30 минут, валидация формы оплаты).
- Обычный HTML/CSS/JS (без сборки) — шаблоны Jinja2, немного vanilla JS
  для интерактива (выбор времени, +/- количества билетов, промокод по AJAX,
  обратный отсчёт таймера).
- Данные (выставки, события, билеты, промокоды) лежат в `data.py` —
  в реальном проекте это была бы таблица в БД.

## Как запустить локально

```bash
cd museum-site
python3 -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Открыть в браузере: **http://localhost:5000**

## Пользовательский путь (как в макете)
1. **/** — главная страница музея.
2. **/exhibition/sequence** — карточка выставки Ричарда Серра, блок
   «Plan your visit»: выбор времени и типа взаимодействия (360° / XR / Guide).
3. **/reservation** — выбор типа и количества билетов + способ связи → «Add to Order».
4. **/checkout** — итоговый заказ, промокод (`MINIMAL10` -10%, `SERRA20` -20%),
   способ оплаты, таймер резерва, кнопка «Pay».
5. **/done** — подтверждение и рекомендации похожих выставок.

## Деплой в интернет (Render)

Проект уже готов к продакшену: `gunicorn` в `requirements.txt`, `Procfile`,
`render.yaml`, секретный ключ и debug-режим читаются из переменных окружения.

1. Загрузите содержимое этой папки в новый репозиторий на GitHub
   (через `git push` или через кнопку **Add file → Upload files** на
   странице пустого репозитория — Git локально не обязателен).
2. На [render.com](https://render.com) → **New +** → **Blueprint** →
   выберите этот репозиторий. Render прочитает `render.yaml` и сам
   создаст веб-сервис на бесплатном тарифе с нужными переменными окружения.
   (Если не хотите использовать Blueprint — можно создать **Web Service**
   вручную: Build Command `pip install -r requirements.txt`,
   Start Command `gunicorn app:app`.)
3. После первого деплоя сайт будет доступен по адресу вида
   `https://museum-of-minimalism.onrender.com`.

Бесплатный тариф Render «засыпает» после 15 минут без запросов и
просыпается около минуты при следующем заходе — для демо-проекта это
не критично.
